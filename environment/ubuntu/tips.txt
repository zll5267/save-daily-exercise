5***************type**********************
type命令的基本使用方式就是直接跟上命令名字。
type -a可以显示所有可能的类型，比如有些命令如pwd是shell内建命令，也可以是外部命令。
type -p只返回外部命令的信息，相当于which命令。
type -f只返回shell函数的信息。
type -t 只返回指定类型的信息
alias 别名
keyword 关键字，Shell保留字
function 函数，Shell函数
builtin 内建命令，Shell内建命令
file 文件，磁盘文件，外部命令
unfound 没有找到
